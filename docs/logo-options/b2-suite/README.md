# TimeAura B2 Logo Suite

这是基于你选中的原始 `B2 / App Icon Ready` 直接延展出来的品牌资产套件。

包含文件：

- `timeaura-b2-app-icon-light.svg`
- `timeaura-b2-app-icon-dark.svg`
- `timeaura-b2-symbol.svg`
- `timeaura-b2-wordmark-darkonlight.svg`
- `timeaura-b2-wordmark-lightondark.svg`
- `timeaura-b2-lockup-horizontal-light.svg`
- `timeaura-b2-lockup-horizontal-dark.svg`
- `timeaura-b2-animation-demo.html`
- `index.html`

建议使用方式：

- 桌面应用图标母版：
  - `timeaura-b2-app-icon-light.svg`
- 小尺寸图形标：
  - `timeaura-b2-symbol.svg`
- 产品页 / 文档封面：
  - `timeaura-b2-lockup-horizontal-light.svg`
- 深色启动页 / 发布物：
  - `timeaura-b2-lockup-horizontal-dark.svg`

当前说明：

- 本轮已把视觉语言收回到原始 B2，不再额外增加 B3 式的光泽和高级徽章感
- 圆弧已升级为完整圆环，适合做“橙点顺时针绕环一周后归位”的时间闭环动画
- 当前先产出 SVG 矢量母版，方便继续细修
- 当前环境没有现成 SVG 位图导出工具，所以还没有直接生成 PNG / ICNS
- 如果下一步要落到 Tauri 图标，我可以继续帮你补：
  - 1024 PNG
  - 512 / 256 / 128 / 64 / 32 PNG
  - macOS `icns`
  - Windows `ico`
