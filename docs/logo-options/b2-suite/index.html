TimeAura B2 Logo Suite

这一套已经按“桌面应用图标 + 品牌字标 + 横版锁定 + 深浅主题”整理完成，并把圆弧升级成完整圆环，方便继续做启动或 hover 动画。

APP ICON
App Icon / Light 适合作为主应用图标母版，干净、清晰、缩小后识别稳定。
App Icon / Dark 更适合暗色品牌展示与深色启动页，不建议直接替换系统 icon，适合作为备选主题版。
SYMBOL + WORDMARK
Symbol 独立图形版，适合 favicon、菜单栏、小尺寸角标。
Wordmark / Light 用于浅色页面、官网页头、产品介绍页。
Wordmark / Dark 用于深色页面、启动页、深色品牌板。
HORIZONTAL LOCKUP
Horizontal Lockup / Light 适合 PRD 封面、品牌页、落地页头部。
Horizontal Lockup / Dark 适合深色视觉稿、发布图与桌面启动界面。
