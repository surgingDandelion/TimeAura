# TimeAura Logo Options

本目录提供 4 个首轮 logo 方向：

- `timeaura-logo-option-a-orbit.svg`
- `timeaura-logo-option-b-monogram.svg`
- `timeaura-logo-option-b1-refined-balanced.svg`
- `timeaura-logo-option-b2-refined-appicon.svg`
- `timeaura-logo-option-b3-refined-premium.svg`
- `timeaura-logo-option-c-beacon.svg`
- `timeaura-logo-option-d-ribbon.svg`

快速预览：

- 打开 `index.html`

建议选择方式：

1. 先选你最想要的品牌气质方向
2. 再决定是否继续细化为：
   - 应用图标版
   - 字标版
   - 深色版
   - 启动画面版
