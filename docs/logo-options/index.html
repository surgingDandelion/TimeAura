TimeAura Logo Options

原始 4 个方向保留，同时新增 3 个基于 B 的精修版。B 系列现在更接近可直接继续落地 App 图标与品牌字标的状态。

B3 / Premium Badge 更偏品牌与高级感，适合官网、启动页和品牌展示。
C / Time Beacon 更偏提醒、信号、智能触达，AI 感更强一点。
D / Aura Ribbon 形态更柔和，品牌化空间大，但时间指向没有 A 明确。
